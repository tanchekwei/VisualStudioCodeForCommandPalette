## Locate `error.log` File

1. Open **File Explorer**

2. Navigate to the following path:

   ```
   C:\Users\<username>\AppData\Local\Packages\15722UsefulApp.WorkspaceLauncherforVSCode_s4b834ss2syh6\LocalState
   ```

3. Replace `<username>` with your Windows username

   * Example:

     ```
     C:\Users\john\AppData\Local\Packages\15722UsefulApp.WorkspaceLauncherforVSCode_s4b834ss2syh6\LocalState
     ```

4. Press **Enter** to open the folder

5. Look for the file:

   ```
   error.log
   ```
